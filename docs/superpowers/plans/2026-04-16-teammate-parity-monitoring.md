# Teammate Parity Monitoring & Gap Analysis Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Instrument Claude Code teammate lifecycle (native Agent-tool teammates + clmux bridge teammates) so we can diff their behaviors on disk, identify gaps, and produce a parity matrix informing future UX / upstream-report work.

**Architecture:** Append-only JSONL event log at `~/.claude/clmux/events.jsonl` fed by (a) Claude Code PostToolUse hooks for `Agent` / `TeamCreate` / `TeamDelete` / `SendMessage` / `TaskCreate` and (b) clmux-side emitters in `update_pane.py` and `clmux-bridge.zsh`. A separate analyzer script produces a markdown parity matrix from the collected events. This is Phase B per Lead's proposal — **observability only, no parity-fixing here**; fixes follow in a separate Phase A/C plan once the matrix shows concrete gaps.

**Tech Stack:** Python 3 stdlib (json, os, sys, time, pathlib, argparse, tempfile), zsh 5, bash 5, Claude Code hooks system (settings.json `hooks.PostToolUse[]`), existing `scripts/_filelock.py`.

**Scope boundary (explicit):** This plan does NOT fix the SendMessage-drop bug, does NOT register bridges into `agentNameRegistry`, does NOT implement rescue/guard hooks. Those are follow-on work gated on the matrix produced here.

```
  teammate-parity-matrix.md   # [generated by Task 9, not hand-written]
settings.json                 # [modified] register PostToolUse hook at repo root
```

**Responsibility split:**
- `_events.py` owns the **schema** and the **durable write** (atomic append with `_filelock` shared lock on the events file). Every emitter goes through this.
- `emit_tool_event.py` owns the **PostToolUse glue** — it parses Claude Code's hook input stream and emits a structured event. It never invents event fields; they all come from the payload.
- `analyze_events.py` owns the **matrix** — what counts as "parity", how per-teammate timelines are reconstructed, what's shown as a gap. Pure read-side, no mutation.
- `update_pane.py` / `clmux-bridge.zsh` only emit events for the **bridge lifecycle points** that have no PostToolUse equivalent (because bridges are spawned outside of Claude Code tools).

**Why not one emitter location:** PostToolUse fires only when Claude invokes a tool. Bridge teammate creation happens through `clmux-bridge.zsh` which runs outside Claude's tool machinery — there is no tool invocation to hook. Keeping the emitter library (`_events.py`) shared lets both sides produce the same schema without duplication.

## Event Schema (canonical — reused by every task)

Each event is one JSON object, one line, appended to `~/.claude/clmux/events.jsonl`:

```json
{
  "ts":           "2026-04-16T00:00:00.000Z",
  "event":        "teammate.spawned",
  "source":       "claude_code | bridge_daemon | update_pane",
  "session_id":   "<claude session id or null>",
  "team_name":    "<team name or null>",
  "teammate":     "<agent name, e.g. plan-reviewer or codex-ci-debug>",
  "agent_id":     "<agentId@team or null>",
  "agent_type":   "teammate | subagent | bridge | general-purpose | null",
  "backend":      "in-process | tmux | external-cli | null",
  "tool":         "<tool name if source=claude_code, else null>",
  "args":         { ...tool args or emitter-specific fields... },
  "result":       { ...tool result or emitter-specific fields... },
  "notes":        "<short free-text for debugging>"
}
```

**Event names (closed set — Task 2 enforces this):**
- `teammate.registered` — Claude Code's `agentNameRegistry.registerName` equivalent moment (derived from Agent tool PostToolUse)
- `teammate.spawned` — tmux pane + CLI became live (bridge-side) or Agent tool returned (native-side)
- `teammate.message_sent` — SendMessage invoked; `args.routing_path` records `qi_ | ZH5 | LH5 | unknown`
- `teammate.message_delivered` — bridge daemon relayed an inbox file line to the tmux pane
- `teammate.state_changed` — isActive true ↔ false transition observed (update_pane writes)
- `teammate.terminated` — tmux pane or subagent task ended
- `team.created`, `team.deleted` — TeamCreate / TeamDelete PostToolUse

Any other event name → dropped by the library with an error log (Task 2 test covers this).

```python
# Append to scripts/_events.py
import datetime
import json
import sys
import tempfile

# Import the project's existing file_lock helper — same pattern other
# clmux scripts use. See scripts/_filelock.py.
_SCRIPTS = Path(__file__).resolve().parent
if str(_SCRIPTS) not in sys.path:
    sys.path.insert(0, str(_SCRIPTS))
from _filelock import file_lock, sigterm_guard  # noqa: E402

ALLOWED_EVENTS = {
    "teammate.registered", "teammate.spawned",
    "teammate.message_sent", "teammate.message_delivered",
    "teammate.state_changed", "teammate.terminated",
    "team.created", "team.deleted",
}

ALLOWED_SOURCES = {"claude_code", "bridge_daemon", "update_pane"}


class EventSchemaError(ValueError):
    """Raised when emit() receives an event or source not in the allowed set."""


def _now_ts() -> str:
    now = datetime.datetime.now(datetime.timezone.utc)
    return now.strftime("%Y-%m-%dT%H:%M:%S.") + f"{now.microsecond // 1000:03d}Z"


def emit(
    event: str,
    source: str,
    teammate,
    agent_type,
    backend,
    tool,
    args: dict,
    result: dict,
    notes: str,
    *,
    session_id=None,
    team_name=None,
    agent_id=None,
) -> None:
    """Append one canonical event to ~/.claude/clmux/events.jsonl.

    Unknown event/source raise EventSchemaError. All other fields
    accept None (null in JSON). See plan "Event Schema" for semantics.
    """
    if event not in ALLOWED_EVENTS:
        raise EventSchemaError(f"unknown event: {event!r}")
    if source not in ALLOWED_SOURCES:
        raise EventSchemaError(f"unknown source: {source!r}")

    ensure_log_dir()
    path = log_path()

    record = {
        "ts": _now_ts(),
        "event": event,
        "source": source,
        "session_id": session_id,
        "team_name": team_name,
        "teammate": teammate,
        "agent_id": agent_id,
        "agent_type": agent_type,
        "backend": backend,
        "tool": tool,
        "args": args,
        "result": result,
        "notes": notes,
    }
    line = json.dumps(record, ensure_ascii=False) + "\n"
    with file_lock(str(path)):
        with sigterm_guard():
            with open(path, "a", encoding="utf-8") as f:
                f.write(line)
```

```python
# hooks/emit_tool_event.py
#!/usr/bin/env python3
"""PostToolUse hook — emit a canonical teammate-lifecycle event.

Claude Code invokes this with a JSON payload on stdin:
  { hook_event_name, session_id, tool_name, tool_input, tool_response }

We only care about five tools. Everything else is a no-op. The hook
MUST NOT fail the agent turn — unknown payloads and write failures
are caught, logged to stderr, and swallowed so exit code stays 0.
"""
import json
import os
import sys
import traceback
from pathlib import Path

# Put scripts/ on path so _events is importable
_HERE = Path(__file__).resolve().parent
_SCRIPTS = _HERE.parent / "scripts"
sys.path.insert(0, str(_SCRIPTS))


def _safe():
    raw = sys.stdin.read()
    try:
        payload = json.loads(raw)
    except Exception:
        print(f"[emit_tool_event] malformed JSON input; bytes={len(raw)}", file=sys.stderr)
        return

    tool = payload.get("tool_name")
    if tool not in ("TeamCreate", "TeamDelete", "Agent", "SendMessage", "TaskCreate"):
        return

    try:
        import _events
    except Exception as e:
        print(f"[emit_tool_event] _events import failed: {e}", file=sys.stderr)
        return

    session_id = payload.get("session_id")
    tool_input = payload.get("tool_input", {}) or {}
    tool_response = payload.get("tool_response", {}) or {}

    # Dispatch per tool
    if tool == "TeamCreate":
        team_name = tool_input.get("team_name") or tool_response.get("team_name")
        _events.emit(
            event="team.created", source="claude_code",
            teammate=None, agent_type=None, backend=None, tool=tool,
            args=tool_input, result=tool_response, notes="",
            session_id=session_id, team_name=team_name,
        )

    elif tool == "TeamDelete":
        team_name = tool_input.get("team_name")
        _events.emit(
            event="team.deleted", source="claude_code",
            teammate=None, agent_type=None, backend=None, tool=tool,
            args=tool_input, result=tool_response, notes="",
            session_id=session_id, team_name=team_name,
        )

    elif tool == "Agent":
        name = tool_input.get("name") or tool_input.get("subagent_type")
        subagent_type = tool_input.get("subagent_type")
        agent_id = tool_response.get("agent_id")
        # Two logical events: registered (registry update moment) + spawned (task ran)
        _events.emit(
            event="teammate.registered", source="claude_code",
            teammate=name, agent_type=subagent_type, backend="in-process",
            tool=tool, args=tool_input, result=tool_response,
            notes="", session_id=session_id, agent_id=agent_id,
        )
        _events.emit(
            event="teammate.spawned", source="claude_code",
            teammate=name, agent_type=subagent_type, backend="in-process",
            tool=tool, args=tool_input, result=tool_response,
            notes="", session_id=session_id, agent_id=agent_id,
        )

    elif tool == "SendMessage":
        to = tool_input.get("to")
        routing = (tool_response.get("routing") or {})
        args = {
            "from": routing.get("sender"),
            "to": to,
            "summary": tool_input.get("summary"),
            "message_chars": len(tool_input.get("message", "")) if isinstance(tool_input.get("message"), str) else None,
            "routing_path": "unknown",  # enriched by analyzer using routing fields
        }
        _events.emit(
            event="teammate.message_sent", source="claude_code",
            teammate=to, agent_type=None, backend=None,
            tool=tool, args=args, result=tool_response,
            notes="", session_id=session_id,
        )

    elif tool == "TaskCreate":
        # We emit for completeness — analyzer may correlate task id with teammate
        _events.emit(
            event="teammate.state_changed", source="claude_code",
            teammate=tool_input.get("assignee"), agent_type=None, backend=None,
            tool=tool, args=tool_input, result=tool_response,
            notes="TaskCreate", session_id=session_id,
        )


try:
    _safe()
except Exception as e:
    print(f"[emit_tool_event] unexpected error: {e}\n{traceback.format_exc()}", file=sys.stderr)

sys.exit(0)
```

```python
#!/usr/bin/env python3
"""Thin CLI wrapper for shell scripts (clmux-bridge.zsh) to emit events
without reimplementing schema validation and locking. Arguments map
directly to _events.emit() keyword args.

Usage:
  _events_zsh_helper.py <event> --source <s> --teammate <t> [--team-name T]
                                [--agent-type A] [--backend B] [--note N]
                                [--pane-id P] [--session-id S]

Silent on success. Prints to stderr on error. Exit code always 0
(observability must not fail the caller).
"""
import argparse
import sys
import traceback
from pathlib import Path

_HERE = Path(__file__).resolve().parent
sys.path.insert(0, str(_HERE))


def main():
    ap = argparse.ArgumentParser()
    ap.add_argument("event")
    ap.add_argument("--source", default="bridge_daemon")
    ap.add_argument("--teammate", default=None)
    ap.add_argument("--team-name", dest="team_name", default=None)
    ap.add_argument("--agent-type", dest="agent_type", default=None)
    ap.add_argument("--backend", default=None)
    ap.add_argument("--note", default="")
    ap.add_argument("--pane-id", dest="pane_id", default=None)
    ap.add_argument("--session-id", dest="session_id", default=None)

try:
    main()
except Exception as e:
```

```python
# scripts/analyze_events.py
#!/usr/bin/env python3
"""Read ~/.claude/clmux/events.jsonl and produce a parity matrix.

Usage:
  analyze_events.py [--input PATH] [--output PATH]

Defaults to reading $HOME/.claude/clmux/events.jsonl and writing to
./teammate-parity-matrix.md.
"""
import argparse
import collections
import datetime
import json
import os
import sys
from pathlib import Path

LIFECYCLE_STEPS = [
    "registered", "spawned", "message_sent", "message_delivered",
    "state_changed", "terminated",
]

GAP_MARK = "—"
PRESENT_MARK = "✓"


def _read_events(path: Path):
    if not path.is_file():
        return []
    out = []
    for line in path.read_text(encoding="utf-8").splitlines():
        if not line.strip():
            continue
        try:
            out.append(json.loads(line))
        except json.JSONDecodeError:
            continue
    return out


def _build_per_teammate(events):
    """Returns {(team, teammate): {step: [event_records]}}."""
    out = collections.defaultdict(lambda: collections.defaultdict(list))
    for e in events:
        team = e.get("team_name") or "_no_team"
        tm = e.get("teammate")
        if tm is None:
            continue
        name = e.get("event", "")
        if not name.startswith("teammate."):
            continue
        step = name.split(".", 1)[1]
        if step in LIFECYCLE_STEPS:
            out[(team, tm)][step].append(e)
    return out


def _count_drops(events, window_seconds=30):
    """Messages sent without a matching delivered within `window_seconds` after."""
    drops = collections.Counter()
    delivered = collections.defaultdict(list)
    for e in events:
        if e.get("event") == "teammate.message_delivered":
            tm = e.get("teammate")
            if tm:
                delivered[tm].append(_parse_ts(e.get("ts")))
    for e in events:
        if e.get("event") != "teammate.message_sent":
            continue
        tm = e.get("teammate")
        if not tm:
            continue
        sent_ts = _parse_ts(e.get("ts"))
        if sent_ts is None:
            continue
        # Match any delivery within window
        matched = any(
            d is not None and 0 <= (d - sent_ts).total_seconds() <= window_seconds
            for d in delivered.get(tm, [])
        )
        if not matched:
            drops[tm] += 1
    return drops


def _parse_ts(s):
    if not s:
        return None
    try:
        return datetime.datetime.fromisoformat(s.replace("Z", "+00:00"))
    except Exception:
        return None


def _render(per_teammate, drops, events):
    lines = ["# Teammate Parity Matrix", ""]
    lines.append(f"Generated: {datetime.datetime.now(datetime.timezone.utc).isoformat()}")
    lines.append(f"Events analyzed: {len(events)}")
    lines.append("")

    # Teams covered
    teams = sorted({t for t, _ in per_teammate.keys()})
    team_events = [e for e in events if e.get("event") in ("team.created", "team.deleted")]
    lines.append(f"Teams observed: {', '.join(teams) if teams else '(none)'}")
    lines.append(f"Team.created/deleted events: {len(team_events)}")
    lines.append("")

    # Per-teammate matrix
    header = ["Team", "Teammate", "Backend"] + LIFECYCLE_STEPS + ["Drops"]
    lines.append("| " + " | ".join(header) + " |")
    lines.append("|" + "|".join(["---"] * len(header)) + "|")
    for (team, tm), steps in sorted(per_teammate.items()):
        # Detect backend from any step's record
        backend = "?"
        for step_events in steps.values():
            if step_events:
                backend = step_events[0].get("backend") or "?"
                break
        row = [team, tm, backend]
        for s in LIFECYCLE_STEPS:
            row.append(PRESENT_MARK if steps.get(s) else GAP_MARK)
        row.append(str(drops.get(tm, 0)))
        lines.append("| " + " | ".join(row) + " |")

    # Drop summary
    lines.append("")
    lines.append("## Message drop summary")
    if drops:
        lines.append("")
        for tm, n in drops.most_common():
            lines.append(f"- **{tm}**: {n} dropped messages (sent but no delivered within 30s)")
    else:
        lines.append("")
        lines.append("No drops detected.")

    # Gap summary
    lines.append("")
    lines.append("## Observed gaps")
    gap_rows = []
    for (team, tm), steps in sorted(per_teammate.items()):
        missing = [s for s in LIFECYCLE_STEPS if not steps.get(s)]
        if missing:
            backend = "?"
            for step_events in steps.values():
                if step_events:
                    backend = step_events[0].get("backend") or "?"
                    break
            gap_rows.append((team, tm, backend, missing))
    if gap_rows:
        for team, tm, backend, missing in gap_rows:
            lines.append(f"- `{team}/{tm}` ({backend}): missing {', '.join(missing)}")
    else:
        lines.append("")
        lines.append("No gaps observed — all teammates hit every lifecycle step.")

    return "\n".join(lines) + "\n"


def main():
    ap = argparse.ArgumentParser()
    ap.add_argument("--input", default=None)
    ap.add_argument("--output", default="teammate-parity-matrix.md")
    args = ap.parse_args()

    if args.input:
        input_path = Path(args.input)
    else:
        input_path = Path(os.environ.get("HOME", str(Path.home()))) / ".claude" / "clmux" / "events.jsonl"

    events = _read_events(input_path)
    per_tm = _build_per_teammate(events)
    drops = _count_drops(events)
    matrix = _render(per_tm, drops, events)

    Path(args.output).write_text(matrix, encoding="utf-8")
    print(f"Wrote {args.output} ({len(events)} events, {len(per_tm)} teammates, {sum(drops.values())} drops)")


if __name__ == "__main__":
    main()
```
